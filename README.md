# 🎓 Grade Master Pro

**Grade Master Pro** adalah aplikasi berbasis web sederhana yang dirancang untuk membantu dosen maupun mahasiswa dalam menghitung dan menentukan nilai akhir secara otomatis. Dengan antarmuka yang interaktif, aplikasi ini memberikan visualisasi data nilai yang menarik dan mudah dipahami.

---

## 🚀 Fitur Utama

* **Perhitungan Berbobot Otomatis**: Mengkalkulasi nilai akhir berdasarkan standar bobot akademik:
    * Tugas: **20%**
    * UTS: **30%**
    * UAS: **50%**
* **Sistem Klasifikasi Grade**: Menggunakan logika *If-Then* untuk menentukan Grade (A, B, C, D, E) secara real-time.
* **UI/UX Modern**: 
    * **Sidebar Input**: Pengaturan nilai menggunakan slider yang intuitif.
    * **Progress Bar**: Visualisasi pencapaian nilai secara grafis.
    * **Dynamic Feedback**: Kartu status kelulusan yang berubah warna sesuai hasil akhir.
* **Efek Interaktif**: Animasi selebrasi (balloons) untuk pencapaian hasil yang memuaskan.

---
